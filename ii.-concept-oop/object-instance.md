description: >-
  วัตถุ (object) หรืออินสแตนซ์ (instance) คือเอนทิตีที่มีชุดคุณสมบัติ
  (attributes) และเมธอด (methods) ของตนเอง
  ซึ่งเราสามารถสร้างวัตถุจากคลาสที่เรากำหนดไว้

# Object / Instance

### Java

```java
Animal a = new Animal();
```

### Python

```python
a = Animal()
```

### C++

```cpp
Animal a;
```
